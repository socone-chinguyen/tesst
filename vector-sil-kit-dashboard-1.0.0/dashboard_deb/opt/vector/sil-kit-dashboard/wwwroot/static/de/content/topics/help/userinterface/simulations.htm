Simulationen

 

Die Seite Simulationen ist die Startseite von SIL Kit Dashboard. Sie gibt einen Überblick über alle Simulationen mit ihrem aktuellen Status und ermöglicht die Auswahl einer Simulation für eine detaillierte Analyse.

Der Inhaltsbereich der Seite Simulationen ist wie folgt aufgebaut:

[1] Tabellenkopf

Der Tabellenkopf beinhaltet konfigurierbare Spalten, in denen die Informationen über die Simulationen angezeigt werden. Informationen über die verfügbaren Spalten finden Sie unten im Abschnitt Tabellendaten.

Um die Spalten zu konfigurieren, klicken Sie auf das Icon einer Spalte. Es öffnet sich ein Menü, in dem Sie Spalten sortieren, filtern oder ein- und ausblenden können.

[2] Tabellendaten

Die Tabellendaten zeigen die Daten der Simulationen an. Die anzuzeigenden Spalten können über den Tabellenkopf konfiguriert werden.

Die folgenden Spalten sind verfügbar:

Gelöscht

Die Spalte Gelöscht ist standardmäßig ausgeblendet. Wenn Simulationsdaten gelöscht wurden, wird hier ein Haken angezeigt, anderenfalls ein leeres Quadrat. Weitere Informationen siehe Spalte Aktionen.

S

Die Spalte S zeigt den Status der Simulationen an:

Erfolgreich Die Simulation wurde erfolgreich durchgeführt und beendet.
Fehlgeschlagen Während der Simulation ist ein Fehler aufgetreten.

Befindet sich die Simulation im Status Läuft, wird ein Spinner-Icon angezeigt.

#

Die Spalte # zeigt die ID der Simulationen an.

Connect URI

Die Spalte Connect URI zeigt die SIL Kit Registry URI der Simulationen an. Über das Suchfeld unterhalb des Spaltennamens kann nach URIs gefiltert werden.

Start

Die Spalte Start zeigt an, wann eine SIL Kit Simulation gestartet wurde.

Ende

Die Spalte Ende zeigt an, wann eine SIL Kit Simulation beendet wurde. Wird eine Simulation noch ausgeführt, ist dieses Feld leer.

Systemstatus

Die Spalte Systemstatus zeigt an, in welchem Status der Zustandsmaschine sich das SIL Kit befindet.

Aktionen

Die Spalte Aktionen stellt die folgenden Optionen bereit:

 Klicken Sie auf dieses Icon, um die Seite Netzwerk-Setup der jeweiligen Simulation zu öffnen.

Klicken Sie auf dieses Icon, um die Seite Zustandshistorie der jeweiligen Simulation zu öffnen.

/

Diese Optionen sind nur für Benutzer mit der Rolle Experte verfügbar.

Klicken Sie auf das Icon , um die Daten der jeweiligen Simulation zu löschen. Nach dem Löschen werden die Simulationsdaten ausgeblendet, bleiben aber während einer definierten Aufbewahrungsfrist, die in den Anwendungseigenschaften des SIL Kit Dashboard Applikationsservers festgelegt ist, zugreifbar.

Um Daten wiederherzustellen, müssen die Tabellenspalten über den Tabellenkopf wie folgt konfiguriert werden:

  1. Aktivieren Sie das Kontrollfeld Spalten|Gelöscht.

  2. Deaktivieren Sie die Kontrollfelder aller Filter.

Die zum Löschen vorgemerkten Simulationsdaten werden wieder angezeigt und das Icon ist verfügbar. Klicken Sie auf , um die entsprechenden Simulationsdaten wiederherzustellen.

[3] Seitenverwaltung

Klicken Sie auf den Pfeil Elemente pro Seite, um auszuwählen, wie viele Elemente pro Seite angezeigt werden sollen.

Um durch die Seiten zu blättern, verwenden Sie die Navigations-Icons.
