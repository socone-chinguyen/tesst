Zustandshistorie

 

Die Seite Zustandshistorie zeigt eine historische Ansicht der Teilnehmerzustände für eine bestimmte Simulation an.

Der Inhaltsbereich der Seite Zustandshistorie ist wie folgt aufgebaut:

[1] Ausgewählte Simulation

In diesem Bereich werden die wichtigsten Fakten über die ausgewählte Simulation angezeigt.

[2] Automatische Aktualisierung der Seite

Wenn sich die Simulation im Zustand Läuft befindet, ist die automatische Aktualisierung der Seite standardmäßig aktiviert, was durch das Icon angezeigt wird.

Um die automatische Aktualisierung der Seite anzuhalten, klicken Sie auf das Icon . Um die automatische Aktualisierung der Seite erneut zu starten, klicken Sie auf das Icon .

[3] Tabellenkopf

Der Tabellenkopf beinhaltet konfigurierbare Spalten, in denen die Informationen über die Zustandshistorie angezeigt werden. Informationen über die verfügbaren Spalten finden Sie unten im Abschnitt Tabellendaten.

Um die Spalten zu konfigurieren, klicken Sie auf das Icon einer Spalte. Es öffnet sich ein Menü, in dem Sie Spalten sortieren oder ein- und ausblenden können.

[4] Tabellendaten

Die Tabellendaten zeigen die Daten der Teilnehmerzustände an. Die anzuzeigenden Spalten können über den Tabellenkopf konfiguriert werden.

Die folgenden Spalten sind verfügbar:

S

Die Spalte S zeigt den Status des Zustandsübergangs an. Im Falle eines Fehlers wird hier ein Icon angezeigt.

Teilnehmername

Die Spalte Teilnehmername zeigt den Namen der Teilnehmer an.

Zustand

Die Spalte Zustand zeigt an, in welchem Zustand der Zustandsmaschine sich ein Teilnehmer befindet.

Eintrittszeit

Die Spalte Eintrittszeit zeigt an, wann ein Teilnehmer in den angezeigten Zustand gewechselt ist.

Grund

Die Spalte Grund zeigt an, warum ein Teilnehmer in den angezeigten Zustand gewechselt ist.

Dauer

Die Spalte Dauer zeigt an, wie lange sich ein Teilnehmer in dem angezeigten Zustand befunden hat.

[5] Seitenverwaltung

Klicken Sie auf den Pfeil Elemente pro Seite, um auszuwählen, wie viele Elemente pro Seite angezeigt werden sollen.

Um durch die Seiten zu blättern, verwenden Sie die Navigations-Icons.
