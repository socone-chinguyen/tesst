Dashboard-Layout

 

Das Dashboard-Layout von SIL Kit Dashboard ist wie folgt aufgebaut:

[1] Navigationsleiste

Die Navigationsleiste bietet die Möglichkeit, zwischen den verschiedenen Seiten von SIL Kit Dashboard zu wechseln.

Nach dem Start von SIL Kit Dashboard ist die Seite Simulationen standardmäßig ausgewählt, da dies der Ausgangspunkt für jede Analyse ist. Sobald in der Spalte Aktionen der Seite Simulationen eine Option ausgewählt wird, zeigt die Navigationsleiste Einträge für zusätzlichen Seiten angezeigt, die Ihnen bei der Analyse Ihrer Simulation helfen können. Über < im Navigationspfad gelangen Sie wieder auf die Seite Simulationen.

Eine Beschreibung der einzelnen Seiten finden Sie in den folgenden Abschnitten:

  • Simulationen

  • Netzwerk-Setup

  • Zustandshistorie

[2] Iconmenü

Das Iconmenü ist auf jeder Seite verfügbar und stellt die folgenden Optionen bereit:

Klicken Sie auf dieses Icon, um ein Menü zu öffnen, das Optionen für weitere Informationen enthält, wie die SIL Kit Dashboard Hilfe, Lizenzinformationen, Nutzungsbedingungen, die Vector Homepage oder die Versionsinformationen von SIL Kit Dashboard.

//

Klicken Sie auf dieses Icon, um ein Menü zu öffnen, in dem Sie zwischen den verfügbaren Sprachen wechseln oder sich aus SIL Kit Dashboard ausloggen können. Das angezeigte Icon hängt von der zugewiesenen Benutzerrolle ab:

  • Anonym (keine Benutzerauthentifizierung)

  • Standardbenutzer

  • Expertenbenutzer

[3] Inhaltsbereich

Je nach aktuell ausgewählter Seite zeigt der Inhaltsbereich die Details des jeweiligen Kontextes an. Weitere Einzelheiten finden Sie in den folgenden Abschnitten:

  • Simulationen

  • Netzwerk-Setup

  • Zustandshistorie
