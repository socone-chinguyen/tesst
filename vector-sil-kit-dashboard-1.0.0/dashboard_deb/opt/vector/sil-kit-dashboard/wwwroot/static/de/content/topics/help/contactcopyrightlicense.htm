Vector Support

Auf unserer Vector Support Website finden Sie unterstützende Antworten aus verschiedenen Quellen für Ihre Arbeit mit Vector Produkten.​

Wenn Sie nicht finden, was Sie suchen, können Sie einen Supportfall eröffnen, um individuelle Hilfe zu erhalten.

  • Vector Support Website (support.vector.com)

  • Vector KnowledgeBase (support.vector.com/kb)

  • Support E-Mail: support@vector.com

Hilfe-Feedback

So geben Sie uns Feedback zur Hilfe:

  1. Erstellen Sie einen Supportfall auf der Vector Support Website; das Feld Betreff ist vorausgefüllt mit Help Feedback.
  2. Kopieren Sie die URL der Hilfeseite, zu der Sie Ihr Feedback senden möchten.
  3. Fügen Sie die URL der Hilfeseite in das Feld Beschreibung ein und schreiben Sie Ihre Kommentare dazu.
  4. Wählen die das Betroffene Produkt aus.
  5. Klicken Sie auf [Weiter].
